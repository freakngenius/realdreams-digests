# RealDreams HQ Digests

This repository stores digest content for the RealDreams HQ - ICYMI dashboard.

## Structure

Each digest is a JSON file with the following format:
- `digest-{date-range}.json`

## Adding a New Digest

1. Create a new JSON file following the naming convention
2. Commit and push to this repository
3. The dashboard will automatically fetch and display it
